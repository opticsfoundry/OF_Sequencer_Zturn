For debugging, this code can run in Visual Studio C++.
Rename all C files into CPP files, e.g. using renameCtoCPP.bat.
Open this directory in Visual Studio C++. 
The CMakeLists.txt should be recognized by CMake and you can compile.

For normal use in Vitis, rename all CPP files into C files, e.g. using renameCPPtoC.bat.
